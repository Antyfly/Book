Тема: Перегрузка методов. Операции и иерархия классов. 


Перегрузка методов.Операции и иерархия классов.

Метод - это функциональный элемент класса. Реализует действия, выполняемые классом/ его экземпляром. 

Перегрузка методов - использование несколько методов с одним и тем же именем, но различными типами и количеством параметров.

Операции методов - перезагружают в основном для классов, с помощью которых задают какие-либо математические понятия, т.е. тогда, когда знаки операции имеют общепринятую семантику. 

+ / - / ! /  ~  (или)   — частичное наследование метода; 
++ / - - — полностью наследует метод;
True / false — используется со значением bool;

Абстрактные классы и интерфейсы. 

Абстрактный класс — это некое обобщение. От этого класса создаются дочерние классы - отряды, и только от них уже создаются объекты. 

Интерфейс — это в большинстве случаев определенная функциональность. Интерфейс может наследовать любым классом. 
В Java объявляется словом abstract. 
При определении интерфейсов вместо class пишется ключевое слово interface. Если есть родительские интерфейсы, они также как в случае классов перечисляются после слова extends. У интерфейсов не может быть родительских классов.
